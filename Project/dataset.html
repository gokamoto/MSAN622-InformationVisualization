Project Dataset

Griffin Okamoto

Description

The data contains information about SF311 cases created since July 1, 2008. Since the original file size was extremely large, I subsetted the data to only include cases from 2015.

Location

The original data set is hosted on SF Open Data, and can be found here. I have also uploaded the subsetted data to my own GitHub repository here.

License Information

Public Domain

Types of Data

The variables in the data set and their corresponding types are listed below.
  • CaseID: numerical/categorical
  • Opened: datetime
  • Closed: datetime
  • Updated: datetime
  • Status: categorical
  • Responsible Agency: categorical
  • Category: categorical
  • Request Type: categorical
  • Request Details: unstructured
  • Address: unstructured
  • Supervisor District: categorical
  • Neighborhood: categorical
  • Point (lat/long): numerical
  • Source: categorical

Why I Chose This Data

I chose this dataset because there are plenty of types of visualizations that are applicable, including geographical and time series. There aren't very many numeric variables, so any numeric information to be encoded will likely be some sort of aggregation. In addition, having moved to San Francisco recently, I'm very interested to see what types of problems are reported in specific areas!
